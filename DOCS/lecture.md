# Целостность и безопасность
## Методы обработки данных. Безопасность данных

Под защитой понимаем несанкционированного или случайного доступа к данным, их изменение или разрушение.

**Можно обеспечить:**
1. Функция безопасности
2. Функция секретности


## Процесс индентификации
Этот процесс выполняется либо с использоваинем системного индентификационного номера, либо TerminalID, ID карта.

## Представление данных
- Ограниченное подмножество, которое создаёт для пользователя личное поле данных.

Пример кода по предоставлению данных:
```SQL
CREATE view dept35
  as select ID, name, address, deptno,age
  from personal
  where deptno = 35;
```

```SQL
GRANT <права доступа> on <на какие таблицы права> to <для кого права>
```

## Уровни доступа
1. Неограниченный доступ ко всему
2. Запрет на доступ к части
3. Доступ к любой части отношения без права изменения содержимого
4. Доступ к одной строке с правом изменения нескольких атрибутов
5. Доступ к одной строке без права изменения содержимого
6. Доступ к отдельным полям с правами отношений. 
7. Доступ к отдельным полям...
